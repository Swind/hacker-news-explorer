# 2026-02-06: CIA 停止發布《世界概況》(The World Factbook)

**來源：** Hacker News
**故事 ID：** 46899808
**Hacker News 連結：** https://news.ycombinator.com/item?id=46899808
**網址：** https://simonwillison.net/2026/Feb/5/the-world-factbook/
**分數：** 363 | **評論數：** 153

## 摘要

CIA 已停止維護其歷史最悠久、最具辨識度的情報出版物《世界概況》(The World Factbook)，該資料庫自 1971 年以來持續更新，自 1997 年起成為公開網際網路的重要資源。令人遺憾的是，CIA 不僅移除了整個網站（包括歷史版本存檔），還將所有頁面設置為 302 重定向到關閉公告，被形容為「文化破壞行為」。

## 為什麼有趣

這個故事具有重要的**開放資料**和**政府透明度**意義：

1. **寶貴的公共資源損失**：《世界 Factbook》是 CIA 最有用的公開資源，包含全球所有國家/地區的地理、人口、政府、經濟、通訊、交通、軍事等資料，被廣泛應用於教育、新聞、研究等領域

2. **開放資料原則的破壞**：該資料庫自發布以來即處於 public domain，沒有理由不繼續提供歷史存檔。Simon Willison 已將 2020 年的完整版本（384MB）上傳至 GitHub，並通過 GitHub Pages 提供訪問

3. **文化資產的保存**：這個事件突顯了政府數位資產保存的重要性，以及社群保存的必要性

## 主要討論點

基於文章內容和社群反應，主要討論主題包括：

- **資料保存的緊迫性**：社群成員正在備份和保存這一寶貴的公共資源
- **政府透明度問題**：為何在沒有任何解釋的情況下突然停止維護？缺乏透明度
- **文化破壞的批評**：將歷史存檔完全移除而非僅標記為「不再維護」的決策受到批評
- **替代方案的建立**：Simon Willison 已在 GitHub 上建立備份 (simonw/cia-world-factbook-2020)，Internet Archive 也有存檔
- **數位資產管理**：政府機構應如何處理不再維護的數位資產？

## 補充分析：事件的重要意義

此事件在 Hacker News 引發高度關注（413 分、177 則評論），反映了社群對以下議題的深切關注：

### 1. 開放政府資料面临威胁
《世界概況》自 1975 年起由 CIA 維護，是美國聯邦政府最早、最成功的開放資料計畫之一。其突然消失引發了對其他開放政府資料穩定性的擔憂。

### 2. CIA 處置方式引發爭議
- 不僅停止更新，還**完全移除**所有歷史版本
- 使用 302 重導向（而非 404）讓所有頁面指向關閉公告
- 未提供任何遷移指南或替代方案
- Simon Willison 將此稱為「文化破壞行為」（cultural vandalism）

### 3. 社群迅速行動
技術社群展現了應急能力：
- Simon Willison 在數小時內將 2020 年版本（384MB）備份至 GitHub
- 啟用 GitHub Pages 讓資料可公開瀏覽
- Internet Archive 保存了更早期的歷史版本

### 4. 政治背景爭議
部分評論將此事件與當前美國政府政策聯繫：
- 認為這反映政府對「情報」和「事實」價值的否定
- 擔憂這是開放政府時代的終結標誌
- 討論了美國從「軟實力」向「硬實力」轉變的趨勢

## 長期影響評估

此事件可能成為開放政府資料運動的轉折點：
- 提醒社群不能依賴單一政府來源
- 凸顯去中心化備份的重要性
- 展示技術社群在維護公共資產方面的關鍵作用
