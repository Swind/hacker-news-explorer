# 2026-02-03: TSA 新的 45 美元無證飛行費用是非法的

## 來源

- **Hacker News**: [The TSA's New $45 Fee to Fly Without ID Is Illegal](https://news.ycombinator.com/item?id=46863162)
- **原文**: [Frommer's - The TSA's New $45 Fee to Fly Without ID Is Illegal, Says Regulatory Expert](https://www.frommers.com/tips/airfare/the-tsa-new-45-fee-to-fly-without-id-is-illegal-says-regulatory-expert/)
- **數據**: 289 點讚，303 則評論

## 摘要

運輸安全管理局（TSA）推出了一項新政策，允許沒有攜帶身份證件的旅客支付 45 美元費用進行身份驗證後登機。監管專家指出，此舉可能違反美國聯邦法律，因為 TSA 並未被授權收取此類費用。該政策引發了關於行政機構權力邊界、隱私權和低收入旅客權益的廣泛討論。

## 為什麼有趣

1. **法律爭議**：聯邦機構未經國會授權單方面徵收新費用，涉及三權分立的核心原則
2. **政策影響廣泛**：每年數百萬旅客可能受影響，特別是經常遺失證件或身分證過期的群體
3. **監管挑戰**：突顯了行政機規則制定程序的透明度和問責問題
4. **社會公義考量**：45 美元費用對低收入旅客構成不成比例的負擔

## 主要討論點

### 法律與監管問題
- **國會授權爭議**：TSA 是否有法定權限收取此項費用？根據美國法律，行政機構新增費用通常需要國會明確授權
- **程序正當性**：費用政策制定過程是否遵守了行政程序法的公告和評論期要求
- **司法審查可能性**：多數評論認為此政策很可能面臨法律挑戰

### 隱私與身份驗證
- **生物識別技術**：45 美元費用是否涉及使用生物識別數據進行身份驗證
- **數據安全擔憂**：第三方身份驗證公司如何處理敏感個人信息
- **監控國家擴張**：此政策是否為更廣泛監控基礎設施的一部分

### 經濟與社會影響
- **累退性影響**：固定費用對低收入旅客造成更大比例負擔
- **替代方案**：臨時駕駛執照、護照等替代證件的取得成本和時間
- **市場壟斷擔憂**：TSA 作為航空安全壟斷提供者的定價權力

### 行政裁量權
- **機構權力邊界**：行政機構何時越過了執法與立法的分界線
- **問責機制**：誰能監督和制衡 TSA 的政策決策
- **先例風險**：其他機構可能效仿此做法擴大收費權

### 實用性與解決方案
- **證件遺失場景**：忘記攜帶 ID 與完全沒有 ID 的區別
- **現有流程**：TSA 已有替代身份驗證流程（如問答驗證），新費用是否必要
- **技術解決方案**：數位身份證和移動驗證的潛在應用
