# HN Arcade 项目分析

## 项目概述

**HN Arcade** 是一个创意十足的项目，将 Hacker News 社区转化为一个游戏化的街机平台。该项目由 Andrew Gy8 开发，旨在收集和展示那些在 HN 社区中分享的创意游戏项目。

## 核心功能

### 1. 游戏发现平台
- 从 Hacker News 上收集各种游戏项目
- 提供友好的浏览界面，让用户发现有趣的 HN 游戏
- 支持按标签分类浏览

### 2. 游戏提交系统
- 允许开发者提交自己的游戏
- 建立了一个专门的游戏目录，方便社区成员发现和参与

## 技术实现

### 架构设计
- **静态网站**: 使用 GitHub Pages 托管
- **游戏聚合**: 自动或手动收集 HN 上的游戏链接
- **标签系统**: 帮助用户按类型过滤游戏

### 技术栈
- 前端：现代 Web 技术
- 托管：GitHub Pages
- 数据管理：基于 JSON 的游戏数据存储
